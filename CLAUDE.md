# Claude Code Entry Point

`AGENTS.md` is imported above. It is the canonical rule file for this
repository.

Use `AGENT_GUIDE.md` for the detailed H-DOCX editing workflow.

Use the `hdocx` MCP server when it is configured. It exposes DOCX inspection,
H-DOCX export, controlled editing, validation, pressure testing, and acceptance
reporting through tools such as `hdocx_audit`, `hdocx_export`,
`hdocx_apply`, `hdocx_diff`, and `hdocx_check`.

Do not create a separate rule interpretation here; keep Codex and Claude Code
aligned on the same contract.
